# Summary
该插件用于检测FLIR应用是否存在目录遍历漏洞，通过尝试访问敏感文件判断漏洞是否存在。。This plugin is used to detect whether the FLIR application has a path traversal vulnerability by attempting to access sensitive files to determine if the vulnerability exists.
