# Summary
该插件用于检测SaltStack Salt中存在的路径名限制不恰当（路径遍历）漏洞，通过发送特定请求判断目标是否存在该漏洞。。This plugin is used to detect the path traversal vulnerability caused by improper path name restrictions in SaltStack Salt by sending specific requests to determine if the target is vulnerable.
