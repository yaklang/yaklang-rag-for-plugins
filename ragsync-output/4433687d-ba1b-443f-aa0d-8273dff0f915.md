# Summary
该插件通过设置多种HTTP请求头（如X-Forwarded-For等）伪造请求IP，以绕过七层安全设备的检测。。This plugin forges the request IP by setting various HTTP headers (such as X-Forwarded-For) to bypass detection of layer 7 security devices.
